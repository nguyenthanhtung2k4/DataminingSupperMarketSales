# Notebooks

Notebook trong repo dung de trinh bay lai pipeline va ket qua.

Logic chinh nam trong `src/`, khong nam trong notebook.

## Thu tu notebook

1. `01_eda.ipynb`
2. `02_preprocess_feature.ipynb`
3. `03_mining.ipynb`
4. `04_modeling_forecasting.ipynb`
5. `05_evaluation_report.ipynb`

## Vai tro tung notebook

- `01_eda.ipynb`: kham pha du lieu
- `02_preprocess_feature.ipynb`: lam sach du lieu va tao dac trung
- `03_mining.ipynb`: association + clustering
- `04_modeling_forecasting.ipynb`: classification + forecasting
- `05_evaluation_report.ipynb`: tong hop ket qua va artifact

## Script lien quan

- `run_pipeline.py`: sinh output chinh
- `run_papermill.py`: chay lai tat ca notebook va ghi output truc tiep vao file trong `notebooks/`
