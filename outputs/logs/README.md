# Logs

Thu muc nay chua log theo tung lan chay.

## Tung file dung de lam gi?

- `best_runs_summary.md`: tong hop run tot nhat hien tai theo tung task
- `run_YYYYMMDD_HHMMSS_<task>.json`: log chi tiet cua tung lan chay

## Script nao tao ra?

- `run_pipeline.py`
- `run_experiments.py`

## Ghi chu

- `run_*.json` chu yeu dung de debug va truy vet.
- Khi can doc nhanh ket qua, uu tien xem `best_runs_summary.md`.
