# Latext

Thu muc nay dung de viet bao cao thu cong.

## Nguyen tac

- Thu muc nay tach khoi luong chay code.
- Pipeline khong tao hoac ghi de file trong day.
- Ban co the viet `thesis.tex` tai day sau khi da co day du ket qua trong `outputs/`.

## File hien co

- `thesis.tex`: file bao cao LaTeX de viet thu cong
- `refs.bib`: tai lieu tham khao neu can
- `figs/`: cac hinh duoc copy tu `outputs/figures/` de phuc vu compile LaTeX

## Nguon du lieu de viet bao cao

Khi can chen ket qua vao bao cao, hay lay tu:

- `outputs/tables/core/`
- `outputs/tables/experiments/`
- `outputs/figures/`
- `outputs/reports/RESULTS_LOG.md`
- `outputs/logs/best_runs_summary.md`

Luu y:

- Khi can dua hinh vao bao cao, nen copy tu `outputs/figures/` sang `docs/Latext/figs/`.
- Khong di chuyen hoac sua file goc trong `outputs/figures/`.
